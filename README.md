# 🟦 PIXELIZER — image & video

이미지·영상을 **픽셀아트**로 바꾸는 브라우저 툴. 그리드 정렬 축소 + 제한 팔레트(메디안컷) + 디더·채도.
모든 처리는 **브라우저 안에서만** — 업로드 없음. 단일 `index.html`(모델·서버 불필요).

**사용:** https://orylo.github.io/image-video-pixelizer/

- **이미지**: 불러오기/드래그 → 픽셀 크기·색상 수·채도·디더 실시간 → 이 장 저장(png) / 전체 저장(zip)
- **영상**: 영상 선택 → 설정 후 **🎬 내보내기** (실시간 녹화, 오디오 포함, webm 출력 — 짧은 클립용)
- 다국어 EN·KO·JA·ZH, 큐 드래그앤드롭, 디자인은 자매 툴 DON'T LOOK HERE와 동일 톤.

## 조작
- 픽셀 크기↑ = 더 굵은 픽셀 / 색상 수↓ = 더 레트로 / 채도 = 색감 / 디더 = 그라데이션 점박이
- 👁 원본 = 원본 미리보기 토글

## 메모
- 영상은 브라우저 실시간 녹화 방식이라 **재생 길이만큼 시간**이 걸리고 출력은 **webm**. 긴 영상·mp4·더 빠른 인코딩이 필요하면 로컬 파이썬(`pixelize_video.py`) 사용.
